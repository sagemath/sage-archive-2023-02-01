= gf2x =

== Description ==
gf2x is a C/C++ software package containing routines for fast arithmetic
in GF(2)[x] (multiplication, squaring, GCD) and searching for
irreducible/primitive trinomials.

Website: http://gf2x.gforge.inria.fr/
